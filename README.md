# 🧠 Воркшоп: Разработка вчера и сегодня — ИИ + сайт за 1.5 часа

Презентация-воркшоп для студентов **НИУ ВШЭ**, посвящённая трансформации процесса разработки в эпоху генеративного ИИ.

## 📖 О проекте

Интерактивная HTML-презентация (14 слайдов), построенная как полноценный практический мастер-класс. Участники проходят полный цикл — от сырой идеи до опубликованного сайта на GitHub Pages — используя ChatGPT / Claude / DeepSeek как ассистента.

## 🎯 Цель воркшопа

Показать студентам, что в 2025 году главная компетенция разработчика — не написание кода, а **управление контекстом**: умение формулировать задачу, создавать документацию и итерировать вместе с ИИ.
